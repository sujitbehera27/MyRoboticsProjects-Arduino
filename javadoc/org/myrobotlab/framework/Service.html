    public abstract class Service
extends java.lang.Object
implements java.lang.Runnable, java.io.Serializable, ServiceInterface
    Service is the base of the MyRobotLab Service Oriented Architecture. All meaningful Services derive from the Service class. There is a _TemplateService.java in the org.myrobotlab.service package. This can be used as a very fast template for creating new Services. Each Service begins with two threads One is for the "OutBox" this delivers messages out of the Service. The other is the "InBox" thread which processes all incoming messages.
