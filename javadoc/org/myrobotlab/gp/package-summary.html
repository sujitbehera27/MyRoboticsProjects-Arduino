Package org.myrobotlab.gp

  • Interface Summary 
    Interface Description
    Validatable
    This interface can be implemented for extensions of user interface components (like TextField or List) whose contents must be checked for validity before they are accepted.
  • Class Summary 
    Class Description
    Choices
    Combines model and view for a choice.
    GP
    This class demonstrates symbolic regression with genetic programming.
    GPMessage
    This is the base class for all the messages that a GP object (an Observable) can send to its observers.
    GPMessageBestFound
    A GP object sends this message to its observers whenever the genetic algorithm has found a new best individual.
    GPMessageEvaluatingIndividual
    A GP object sends this message to its observers whenever the genetic algorithm has evaluated the fitness of an individual.
    GPMessageEvaluatingPopulation
    A GP object sends this message to its observers whenever the genetic algorithm has evaluated the fitnesses of all individuals in the population.
    GPMessageFitnessCasesSet
    A GP object sends this message to its observers whenever the fitness cases change.
    GPMessageStateChanged
    A GP object sends this message to its observers whenever the execution state of the genetic algorithm changes, such as from "Running" to "Stopped".
