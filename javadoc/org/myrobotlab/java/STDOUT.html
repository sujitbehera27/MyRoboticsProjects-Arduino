    public class STDOUT
extends javax.swing.JFrame
implements StreamObserver
    Class STDOUT creates a Java Console for GUIService based Java Applications. Once created, a Console component receives all the data directed to the standard output (System.out) and error (System.err) streams.

    For example, once a Java Console is created for an application, data passed on to any methods of System.out (e.g., System.out.println(" ")) and System.err (e.g., stack trace in case of uncought exceptions) will be received by the Console.

    Note that a Java Console can not be created for Applets to run on any browsers due to security violations. Browsers will not let standard output and error streams be redicted (for obvious reasons).
