Class HTTPRequest

  • java.lang.Object
    • org.myrobotlab.net.HTTPRequest

  • public class HTTPRequest
extends java.lang.Object
    Client HTTP Request class This class helps to send POST HTTP requests with various form data, including files. Cookies can be added to be included in the request.
    Version:
    1.0 Modified by grog - added buffered output to increase performance in larger POSTs. In file operations buffered output is 10x faster.
