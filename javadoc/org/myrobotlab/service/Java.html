      • preProcessHook

        public boolean preProcessHook(Message msg)
        preProcessHook is used to intercept messages and process or route them before being processed/invoked in the Service. Here all messages allowed to go and effect the Java service will be let through. However, all messsages not found in this filter will go "into" they Java script. There they can be handled in the scripted users code.
