    public class TranscriptionThread
extends java.lang.Thread
    Each transcription is handled in a separate thread to ensure that the program remains responsive while the thread is waiting for a response from the server. Lifted from a great page http://stt.getflourish.com/ by Mr. Florian Schulz. More info in Chromium source :) http://src.chromium.org/svn/trunk/src/ content/browser/speech/speech_recognition_request.cc
